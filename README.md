# Almail+ Official 1.0.0 - MELI Ecosystem Support

> 📘 This section is in English. For the Portuguese version, scroll down to the section marked "Versão em Português".

**Almail+** is an intelligent web application designed to empower **Mercado Libre and Mercado Pago** support teams with AI-powered assistance. Built specifically for the MELI ecosystem, it provides specialized guidance for customer service representatives across multiple channels and business units.

## 🚀 Key Features

### 🤖 **Ecosystem Intelligence**
* **Smart Ecosystem Recognition**: Automatically adapts responses based on Mercado Pago, Mercado Libre, or SOS contexts
* **Specialized AI Responses**: Context-aware assistance tailored to each business unit's specific needs
* **SOS Restricted Mode**: Secure, limited-scope responses for emergency support channels

### 💬 **Advanced Chat Interface**
* Modern, responsive chat interface optimized for customer support workflows
* **Persistent Conversation History**: All conversations securely stored in Firebase Firestore
* **Smart Conversation Management**: Create, edit, delete, and organize support conversations
* **Real-time Response Generation**: Dynamic AI responses with typing animations

### 🎨 **Enhanced User Experience**
* **Welcome Modal**: Interactive onboarding experience showcasing new features
* **Modern Credits System**: Beautiful team recognition with GitHub integration
* **Dark/Light Theme Toggle**: Seamless theme switching with system preference detection
* **Responsive Design**: Optimized for desktop, tablet, and mobile devices

### 🔒 **Security & Performance**
* **Anonymous Authentication**: Secure Firebase authentication without personal data collection
* **Performance Optimized**: Fast loading times and efficient resource management
* **Production Ready**: Built with Vite for optimal performance and modern web standards

## 🆕 What's New in Official 1.0.0

### **Major Upgrades from Beta 1.4.0:**

- **🔄 Complete Architecture Rewrite**: Migrated from vanilla JS to React with modern tooling
- **🏢 Ecosystem Intelligence**: AI now recognizes and adapts to MP, ML, and SOS environments
- **🛡️ SOS Security Mode**: Restricted AI responses for emergency support channels
- **⚡ Performance Boost**: 3x faster loading times with optimized bundling
- **🎨 Modern UI/UX**: Complete visual overhaul with consistent design language
- **📱 Enhanced Responsiveness**: Better mobile and tablet support
- **🔐 Improved Security**: Enhanced authentication and data protection
- **🌟 Welcome Experience**: Interactive onboarding for new users
- **👥 Team Recognition**: Beautiful credits system with developer profiles

### **Technical Improvements:**
- React 18 with modern hooks and state management
- Vite build system for lightning-fast development
- Tailwind CSS for consistent styling
- Firebase 10 with improved security rules
- ESLint integration for code quality
- Modern browser compatibility

## ⚙️ Setup and Development

### **Prerequisites**
* Node.js 18+ and npm
* Modern web browser
* Firebase project (for production deployment)

### **Installation**
```bash
git clone https://github.com/boltreskh/almail.git
cd almail
npm install
```

### **Development**
```bash
npm run dev
# Opens development server at http://localhost:5173
```

### **Production Build**
```bash
npm run build
npm run preview  # Preview production build
```

### **Code Quality**
```bash
npm run lint     # Run ESLint
```

# Almail+ Oficial 1.0.0 - Suporte Ecossistema MELI

> 📘 Esta seção está em português. Para a versão em inglês, volte para o topo ou procure pela seção "English Version".

**Almail+** é uma aplicação web inteligente projetada para capacitar as equipes de suporte do **Mercado Livre e Mercado Pago** com assistência baseada em IA. Construído especificamente para o ecossistema MELI, fornece orientações especializadas para representantes de atendimento ao cliente em múltiplos canais e unidades de negócio.

## 🚀 Principais Funcionalidades

### 🤖 **Inteligência de Ecossistema**
* **Reconhecimento Inteligente**: Adapta automaticamente as respostas baseado nos contextos Mercado Pago, Mercado Livre ou SOS
* **Respostas IA Especializadas**: Assistência contextual personalizada para as necessidades específicas de cada unidade de negócio
* **Modo SOS Restrito**: Respostas seguras e limitadas para canais de suporte de emergência

### 💬 **Interface de Chat Avançada**
* Interface de chat moderna e responsiva otimizada para fluxos de trabalho de suporte ao cliente
* **Histórico Persistente**: Todas as conversas armazenadas com segurança no Firebase Firestore
* **Gerenciamento Inteligente**: Criar, editar, excluir e organizar conversas de suporte
* **Geração de Respostas em Tempo Real**: Respostas dinâmicas de IA com animações de digitação

### 🎨 **Experiência do Usuário Aprimorada**
* **Modal de Boas-Vindas**: Experiência interativa de onboarding apresentando novas funcionalidades
* **Sistema de Créditos Moderno**: Belo reconhecimento da equipe com integração GitHub
* **Alternância de Tema Claro/Escuro**: Troca suave de temas com detecção de preferência do sistema
* **Design Responsivo**: Otimizado para desktop, tablet e dispositivos móveis

### 🔒 **Segurança e Performance**
* **Autenticação Anônima**: Autenticação Firebase segura sem coleta de dados pessoais
* **Performance Otimizada**: Tempos de carregamento rápidos e gerenciamento eficiente de recursos
* **Pronto para Produção**: Construído com Vite para performance otimizada e padrões web modernos
